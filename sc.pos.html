tb.sc.pos¶

Utilities to choose a list of random positions with the given number of dimensions dim and of the given length size from a uniform grid with the given shape.

Classes¶

class tb.sc.pos.PosRule(*args, **kwargs)[source]¶
class tb.sc.pos.all(shape=None, mix=None, num=None)[source]¶

Simply get all positions of a lattice with the given shape. So, there is no randomness included, hence this class is (as you may think) very simple in nature.

class tb.sc.pos.hom(shape=None, mix=None, num=None)[source]¶

Generate random positions (integer tuples) of a lattice with the given shape. It is made sure that no position is generated twice. The positions are distributed homogeneously, i.e. every possible position is chosen with the same probability.

Call an instance of this class, providing the shape of the lattice and the number of positions size needed.

class tb.sc.pos.spheres(shape=None, rad=1.0, space=0.0, sconc=None, iconc=None, iconcin=None, iconcout=None, timeout='10s')[source]¶

Generate random positions (integer tuples) of a lattice with the given shape under the existence of spherical imhomogeneities.

It is made sure that no position is generated twice.

Call an instance of this class, providing the shape of the lattice and the number of positions size needed.

Find the number of points of a dim-dimensional uniform grid that are located within a sphere with the given radius, assuming the sphere itself is centred on one of the grid points.
